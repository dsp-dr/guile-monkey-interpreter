# Chapter 4: Extending the Interpreter

This chapter adds arrays, hashes, and built-in functions.

## New Features

- Arrays: `[1, 2, 3]`
- Hashes: `{"key": "value"}`
- Built-in functions: `len`, `first`, `last`, `rest`, `push`, `puts`

## Running

```bash
guile -L src/monkey --no-auto-compile -l main.scm
```
